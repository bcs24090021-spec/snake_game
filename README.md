# Snake Game

A minimal console-based Snake game implemented in Python using curses.

## Run
- Ensure Python 3.x is installed
- On Windows, install curses support if needed: `pip install windows-curses`
- Run: `python snake.py`

## Controls
- Arrow keys or WASD to move
- Q to quit

## Notes
- The game centers in the terminal; adjust your window size if needed.
